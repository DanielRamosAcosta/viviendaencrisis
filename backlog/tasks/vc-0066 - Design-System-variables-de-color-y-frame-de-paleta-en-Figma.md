## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Crear las variables de color (primitivas + tokens semánticos) en el fichero Main Design de Figma, y un frame de exposición en la página 🎨 Design System que muestre la paleta constructivista del proyecto.
<!-- SECTION:DESCRIPTION:END -->

## Final Summary

<!-- SECTION:FINAL_SUMMARY:BEGIN -->
Variables de color creadas en dos colecciones: "Primitivos" (6 vars: rojo, negro, crema, blanco-roto, gris-oscuro, verde) y "Colores" con 8 tokens semánticos aliased (action/primary|secondary, surface/default|elevated, text/primary|secondary, feedback/error|success). Frame `color-palette` creado en la página 🎨 Design System con header constructivista negro+barra roja, sección de primitivos con swatches 196×197px y sección de tokens 4×2 con color block + nombre + uso + hex.
